Nexoryx installieren

Ein Befehl genügt. Erkanntes System: …

$ curl -fsSL http://192.168.13.100:3007/install.sh | bash

x

01Voraussetzungen

  • Python 3.11+ — prüfen mit python3 --version
  • curl und (empfohlen) git
  • Internet für den Erst-Download der Modelle

02Pro Betriebssystem

🐧 Linux

Terminal öffnen, Befehl einfügen, ausführen. Fertig.

$ curl -fsSL http://192.168.13.100:3007/install.sh | bash

🍎 macOS

Falls Python fehlt: brew install python@3.12, dann:

$ curl -fsSL http://192.168.13.100:3007/install.sh | bash

🪟 Windows

Empfohlen via WSL (Ubuntu) oder Git-Bash. In der WSL-Shell:

$ curl -fsSL http://192.168.13.100:3007/install.sh | bash

03Prüfen

$ python3 -m nexoryx doctor

Zeigt erkannte Hardware, gewähltes Profil und welche Modelle erlaubt sind.

04Telegram (optional)

Während der Installation kannst du einen Bot-Token + deine Telegram-User-ID angeben. Ohne Telegram: bei der Abfrage einfach überspringen (oder --no-telegram).

05Hilfe & Troubleshooting

  • python3 nicht gefunden → Python 3.11+ installieren, Terminal neu öffnen.
  • Modell zu groß / langsam → normal auf schwacher Hardware; Nexoryx nutzt automatisch das kleinste passende Modell oder die Cloud.
  • Logs liegen unter ~/.nexoryx/.
Hinweis: Installation über 192.168.13.100 = Admin-Modus. Aus dem öffentlichen Repo installiert = User-Modus (ohne Admin-Funktionen).
