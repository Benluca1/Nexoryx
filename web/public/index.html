$ curl -fsSL http://192.168.13.100:3007/install.sh | bash

Erkanntes System: … · von diesem Server installiert = Admin-Modus 🔑

01Läuft das auf meinem Gerät?

Nexoryx passt sich automatisch an. Drei Stufen, je nach Hardware:

Ultra-Lite CPU-only

Alte Laptops, ab ~4 GB RAM. Kleines lokales Modell, sehr sparsam.

Balanced 8 GB+

Normale Rechner. Kleine + mittlere Modelle, optional Cloud.

Pro GPU

Starke GPU (≥24 GB VRAM) oder ≥48 GB RAM. Große Modelle, Multi-Agent.

Eigene, gestaffelte Modell-Familie — automatisch passend installiert.

Für absolute Spitzenqualität routet Nexoryx automatisch in die Cloud (Anthropic, OpenAI, Google) — du gibst die Keys bei der Installation an.

03In 3 Schritten startklar

  1. Befehl kopieren (oben) und im Terminal einfügen.
  2. Fragen beantworten — Nexoryx prüft die Hardware und fragt optional nach API-Keys & Telegram.
  3. Loslegen: nexoryx ask "Hallo" oder über den Telegram-Bot.

Ausführliche Schritt-für-Schritt-Anleitung pro Betriebssystem → install.html

04Wichtige Befehle

nexoryx doctorHardware + Profil prüfen
nexoryx ask "…"Eine Frage stellen (Router → lokal/Cloud)
nexoryx run "…"Aufgabe über Planner + Agenten
nexoryx exec "…"Shell in Sandbox (Security + Audit)
nexoryx memory …Speicher list/search/forget/remember
nexoryx daemonHTTP-API (nexoryxd) starten
nexoryx telegramTelegram-Bot starten
nexoryx admin …Keys/Rollen/Telegram (nur Server-Install)
nexoryx panicNot-Aus: alles stoppen
Admin-Funktionen nur über diesen Server. Wer Nexoryx über 192.168.13.100 installiert, wird Owner/Admin seines Geräts (Modelle, Keys, Rollen, Logs verwalten). Installationen aus dem öffentlichen Repo laufen im normalen User-Modus.
